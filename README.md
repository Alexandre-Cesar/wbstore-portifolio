# 📱 WB Store | Assistência Técnica Gamer

![Thumbnail](/images/tela_portifolio.png)

Uma Landing Page moderna e responsiva para serviços de assistência técnica em dispositivos móveis, desenvolvida com uma estética "Cyberpunk/Gamer". O projeto destaca reparos de hardware como se fossem "Upgrades" ou "Respawns" de equipamentos.

---

## 🚀 Tecnologias

* **HTML5:** Estrutura semântica das seções.
* **CSS3 (Custom Properties):** Design focado em cores neon (`#00f3ff`, `#9d00ff`), efeitos de brilho e fontes tecnológicas (**Orbitron**).
* **JavaScript (Vanilla):** Implementação de **Scroll Spy** para navegação dinâmica e detecção de seção ativa.

---

## ✨ Funcionalidades

* **Estética Cyber-Tech:** Uso de sombras neon, tipografia futurista e cards com efeitos de hover.
* **Menu Inteligente:** O navbar identifica automaticamente em qual seção o usuário está e destaca o link correspondente.
* **Botão de Contato Direto:** Integração flutuante com WhatsApp para orçamentos rápidos.
* **Scroll Suave:** Navegação fluida entre as seções `START`, `SKILLS` e `CONTATO`.
* **Seções de Serviços (Skills):** Cards estilizados para Troca de Tela, Bateria e Software.
